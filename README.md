## 🛠 الأدوات المتوفرة

### 📅 حاسبة العمر
احسب عمرك بالسنوات والأشهر والأيام بدقة.

### 💰 حاسبة ضريبة القيمة المضافة
حساب ضريبة القيمة المضافة والإجمالي النهائي بسرعة.

### 💵 حاسبة الرواتب
حساب صافي الراتب بعد إضافة البدلات وخصم الاستقطاعات.

### 📊 حاسبة النسبة المئوية
إيجاد النسب المئوية لأي قيمة بسهولة.

### ⚖️ حاسبة مؤشر كتلة الجسم (BMI)
معرفة ما إذا كان الوزن ضمن المعدل الطبيعي.

### 🏦 حاسبة القروض
تقدير قيمة القسط الشهري للقروض.

### 🕌 حاسبة الزكاة
حساب مقدار الزكاة المستحقة.

### 🔐 مولد كلمات المرور
إنشاء كلمات مرور قوية وعشوائية.

### 📄 مولد الفواتير
إنشاء فواتير بسيطة وسريعة.

### 🔗 مولد QR Code
تحويل الروابط والنصوص إلى رموز QR قابلة للمسح.
